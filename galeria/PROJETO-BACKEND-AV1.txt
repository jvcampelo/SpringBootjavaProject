PARA A AV1:

CRIAR UM SISTEMA DE CADASTRO DE GALERIA DE CARROS/MOTOS:
DEVERÁ CONTER PELO MENOS DUAS TABELAS RELACIONADAS FABRICANTE -> VEICULO

AS TABELAS DEVERÁ SER MODELADA COM OS SEGUINTES CAMPOS
FABRICANTE:
 ID, 
 NOME,
 DATA DE CRIAÇÃO

VEICULO:
 ID,
 NOME,
 TIPO (CARRO OU MOTO)
 DATA DE CRIAÇÃO
 DESCRIÇÃO DO VEICULO,
 IMAGEM

UTILIZANDO SQL LITE, CRIE AS TABELAS NO SPRING BOOT, SUAS RELAÇÕES E AS TELAS DE DEVERÁ TER:
CRUD DE FABRICANTE
CRUD DE VEICULO, ASSOCIANDO UMA FABRICANTE UM VEICULO

OS DADOS DEVERÃO SER VALIDADOS NO SISTEMA E INFORMAR AO USUÁRIO A MENSAGEM DE ERRO.
